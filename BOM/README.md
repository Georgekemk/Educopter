# Bill of Materials (BOM)

This Bill of Materials (BOM) provides a complete list of components used throughout the EDUCOPTER project. It includes parts required for both the EDUCOPTER Linux flight controller board and the final quadcopter platform used for testing and demonstration. As such, the list covers everything from electronic components on the flight controller (sensors, connectors, passive components, etc.) to the wider drone system, including items such as the frame, motors, ESCs, power distribution board, receiver, and GPS module. These external drone components are included primarily for completeness; builders do not need to select the exact same frame, motors, ESCs, or power distribution board listed here, and alternative compatible hardware may be used.

Some small electronic components (such as resistors, transistors, and Dupont cables) are commonly sold only in bulk quantities, which may not be ideal when sourcing parts individually. In educational environments or laboratories where basic electronics supplies are already available, it is recommended to use those existing components rather than purchasing large quantities unnecessarily. The notes column within the BOM indicates which parts are actually required in such cases.

This build assumes access to basic electronics assembly tools, specifically a soldering iron and solder, which are required for assembling the board and making electrical connections.

The custom EDUCOPTER circuit board itself is not included in the BOM. It is assumed that users will either order the PCB from a manufacturer using the provided design files, or produce the board themselves using a PCB milling machine (for example, the Makera Carvera).
